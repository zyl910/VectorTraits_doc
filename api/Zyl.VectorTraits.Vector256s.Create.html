Create<T>(Vector128<T>, Vector128<T>)

Create a new vector instance from two shorter vector instances. (从两个更短的向量实例, 来创建新的向量实例).

public static Vector256<T> Create<T>(Vector128<T> lower, Vector128<T> upper) where T : struct

Parameters

lower Vector128<T>

The value that the lower 128-bits will be initialized to (较低 128 位的初始化值).

upper Vector128<T>

The value that the upper 128-bits will be initialized to (较高 128 位的初始化值).

Returns

Vector256<T>

A new Vector256<T> initialized from lower and upper (从 lower 和 upper 初始化的新 Vector256<T>).
