YGroup2Transpose<T>(IVectorTraits, Vector<T>, Vector<T>)

Transpose a 22 matrix (对22矩阵进行转置). Mnemonic: result0[i] := (0==(i&1))?( x[i&~1] ):( y[i&~1] ), result1[i] := (0==(i&1))?( x[(i&~1) + 1] ):( y[(i&~1) + 1] ).
