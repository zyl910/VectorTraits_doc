YGroup2TransposeEven(Vector512<float>, Vector512<float>)

Transpose a 22 matrix and return a data in even positions (对22矩阵进行转置, 并返回偶数位置的数据). Mnemonic: rt[i] =: (0==(i&1))?( x[i&~1] ):( y[i&~1] ).
