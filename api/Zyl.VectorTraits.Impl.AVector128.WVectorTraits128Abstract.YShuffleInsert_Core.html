YShuffleInsert_Core(Vector128<float>, Vector128<float>, Vector128<int>, Vector128<int>, Vector128<int>)

Core calculation for shuffle and insert (换位并插入的核心计算). Its arguments are derived from YShuffleInsert_Args (其参数来源于 YShuffleInsert_Args). Mnemonic: rt[i] := (0<=indices[i] && indices[i]<Count)?( vector[indices[i]] ):back[i].

public virtual Vector128<float> YShuffleInsert_Core(Vector128<float> back, Vector128<float> vector, Vector128<int> args0, Vector128<int> args1, Vector128<int> args2)

Parameters

back Vector128<float>

The background vector (背景向量).

vector Vector128<float>

Arguments 0 (参数0). Derived from YShuffleInsert_Args .

args1 Vector128<int>

Arguments 1 (参数1). Derived from YShuffleInsert_Args .

args2 Vector128<int>

Arguments 2 (参数2). Derived from YShuffleInsert_Args .

Returns

Vector128<float>
