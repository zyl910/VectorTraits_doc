YShuffleX3Kernel_AcceleratedTypes

Types with hardware acceleration when running YShuffleX3Kernel (运行 YShuffleX3Kernel 时具有硬件加速的类型).

public static TypeCodeFlags YShuffleX3Kernel_AcceleratedTypes { get; }

Property Value

TypeCodeFlags

Remarks

Meaning of suffixes (后缀的含义).

- (none): Normal (常规).

- Args: Arguments calculation for only shuffle on 3 vectors (在3个向量上进行仅换位的参数计算). Provide arguments for YShuffleX3Kernel_Core (为 YShuffleX3Kernel_Core 提供参数).

- Core. Core calculation for only shuffle on 3 vectors 在3个向量上进行仅换位的核心计算). Its arguments are derived from YShuffleX3Kernel_Args (其参数来源于 YShuffleX3Kernel_Args).

Similar methods (相似的方法).

- YShuffleX3(Vector<byte>, Vector<byte>, Vector<byte>, Vector<byte>): If the index value is out of range, the element will be cleared (若索引值超出范围, 元素会被清零).

- YShuffleX3Insert(Vector<byte>, Vector<byte>, Vector<byte>, Vector<byte>, Vector<byte>): If the index value is out of range, the elements of the background vector will be inserted (若索引值超出范围, 会插入背景向量的元素).

- YShuffleX3Kernel(Vector<byte>, Vector<byte>, Vector<byte>, Vector<byte>): If the index value is out of range, the result is undefined (若索引值超出范围, 结果是未定义的). It has the best performance, and is used when you are sure that the index value is not out of range (它的性能最好, 用于确信索引值不会超出范围时).
